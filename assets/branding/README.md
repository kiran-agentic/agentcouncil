# AgentCouncil Logo

This logo system is built around the project's core idea: two independent agents form distinct positions, then feed a shared council chamber.

Files:

- `logo-mark.svg`: transparent icon for light backgrounds
- `logo-lockup.svg`: mark plus wordmark and tagline
- `logo-badge.svg`: dark badge variant for avatars and social cards

Palette:

- `#0F172A`: council navy
- `#1A9F93`: agent one / exploration
- `#F59E0B`: agent two / challenge
- `#F7F4EA`: paper / consensus highlight

The open ring creates a subtle `C` for Council, while the two inner wedges imply distinct agent stances feeding a shared decision space.
