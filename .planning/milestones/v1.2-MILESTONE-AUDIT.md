## Tech Debt

### Phase 16: Conformance Certification
- **CERT gate inactive in production:** `check_certification_gate` is called without a `CertificationCache` instance — gate always returns immediately. Fix: inject `CertificationCache()` at the call sites in `server.py`.
- **`_test_tool_use` uses StubProvider-specific `.calls`:** Accesses `provider.calls` which only exists on `StubProvider`. Real providers always return `tool_use=False`. Fix: track invocation count in the certifier itself.
- **`datetime.utcnow()` deprecated:** Use `datetime.now(timezone.utc)` instead.

### Phase 13: OpenRouter Provider
- **`httpx` undeclared dependency:** Used in `openrouter.py:auth_check()` but not in `[openrouter]` extras. Currently available via `openai` transitive dep. Fix: add `httpx>=0.27` to extras.

### Phase 15: MCP Integration
- **Double `ProfileLoader().resolve()` call:** Minor efficiency issue in all skill tools. Fix: cache the resolved profile from `_make_provider`.

**Total: 5 items across 3 phases — none are blockers**
